## Codebase Patterns
- Use `npm install --cache /tmp/npm-cache` to work around root-owned npm cache files in ~/.npm
- Exclude `scripts` directory in tsconfig.json to avoid type errors from embedded git repos
- Use `@/*` path alias for imports from `./src/*`
- Static export via `output: "export"` in next.config.ts
- Tailwind v4 with `@tailwindcss/postcss` plugin (no tailwind.config.js needed)
